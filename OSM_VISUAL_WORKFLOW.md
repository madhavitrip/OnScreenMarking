## Annotation Types & Usage

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        ANNOTATION TYPES                                 │
└─────────────────────────────────────────────────────────────────────────┘

1. PEN (Freehand Drawing)
   ┌─────────────────────────────────────────────────────────────────────┐
   │ Use for:                                                            │
   │ • Marking answers                                                   │
   │ • Writing marks directly                                            │
   │ • Highlighting sections                                             │
   │ • Adding emphasis                                                   │
   │                                                                     │
   │ Example:                                                            │
   │ [Answer text...]                                                    │
   │ ✓ ← Green pen mark                                                  │
   │ "2 marks" ← Written with pen                                        │
   │                                                                     │
   └─────────────────────────────────────────────────────────────────────┘

2. TEXT (Typed Annotations)
   ┌─────────────────────────────────────────────────────────────────────┐
   │ Use for:                                                            │
   │ • Adding explanations                                               │
   │ • Writing marks                                                     │
   │ • Adding feedback                                                   │
   │ • Noting issues                                                     │
   │                                                                     │
   │ Example:                                                            │
   │ [Answer text...]                                                    │
   │ "Good explanation but incomplete" ← Text annotation                 │
   │                                                                     │
   └─────────────────────────────────────────────────────────────────────┘

3. CIRCLE (Shape Tool)
   ┌─────────────────────────────────────────────────────────────────────┐
   │ Use for:                                                            │
   │ • Marking correct answers (green)                                   │
   │ • Marking partial answers (orange)                                  │
   │ • Highlighting important sections                                   │
   │ • Quick visual feedback                                             │
   │                                                                     │
   │ Example:                                                            │
   │ [Answer text...]                                                    │
   │ ⭕ ← Green circle (Correct)                                         │
   │ ⭕ ← Orange circle (Partial)                                        │
   │                                                                     │
   └─────────────────────────────────────────────────────────────────────┘

4. SQUARE (Shape Tool)
   ┌─────────────────────────────────────────────────────────────────────┐
   │ Use for:                                                            │
   │ • Highlighting sections                                             │
   │ • Marking areas for review                                          │
   │ • Organizing annotations                                            │
   │ • Grouping related content                                          │
   │                                                                     │
   │ Example:                                                            │
   │ ┌─────────────────────────────────────────────────────────────────┐ │
   │ │ [Answer text...]                                                │ │
   │ │ [Important section highlighted with square]                    │ │
   │ └─────────────────────────────────────────────────────────────────┘ │
   │                                                                     │
   └─────────────────────────────────────────────────────────────────────┘

5. ERASER (Removal Tool)
   ┌─────────────────────────────────────────────────────────────────────┐
   │ Use for:                                                            │
   │ • Removing mistakes                                                 │
   │ • Cleaning up annotations                                           │
   │ • Correcting errors                                                 │
   │ • Precise editing                                                   │
   │                                                                     │
   │ Example:                                                            │
   │ [Answer text...]                                                    │
   │ ✓ ← Erased incorrect mark                                           │
   │ ✗ ← New correct mark                                                │
   │                                                                     │
   └─────────────────────────────────────────────────────────────────────┘
```

## Color Coding System

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        COLOR MEANINGS                                   │
└─────────────────────────────────────────────────────────────────────────┘

🟢 GREEN
   Meaning: Correct / Full Marks
   Usage: Mark correct answers with green circle or pen
   Example: ✓ Q1 - Correct answer

🔴 RED
   Meaning: Wrong / No Marks
   Usage: Mark wrong answers with red cross or pen
   Example: ✗ Q2 - Wrong answer

🟠 ORANGE
   Meaning: Partial / Partial Marks
   Usage: Mark partial answers with orange circle
   Example: ⚠ Q3 - Partial credit

🔵 BLUE
   Meaning: Neutral / Information
   Usage: General annotations and notes
   Example: "See page 2 for continuation"

🟡 YELLOW
   Meaning: Warning / Attention
   Usage: Mark areas needing review
   Example: Highlight unclear handwriting

⚫ BLACK
   Meaning: Standard / Default
   Usage: General writing and marks
   Example: "2 marks" written on sheet
```

## Keyboard Shortcuts

```
┌─────────────────────────────────────────────────────────────────────────┐
│                      KEYBOARD SHORTCUTS                                 │
└─────────────────────────────────────────────────────────────────────────┘

P    → Pen Tool
T    → Text Tool
C    → Circle Tool
S    → Square Tool
E    → Eraser Tool

Ctrl+Z → Undo (if supported)
Ctrl+C → Copy Selected Text
Delete → Clear All Annotations

Arrow Keys → Navigate Pages
Space → Toggle Annotation Panel
Esc → Close Modals
```
