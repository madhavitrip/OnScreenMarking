## 🔄 Registration Flow

```
1. User submits registration
2. AuthController validates input
3. Generate password using PasswordGenerationService
4. Create user with auto-generated password
5. Save to database
6. Send email with credentials
7. Return JWT token
```

## 📧 Email Integration

### Three Methods Available

1. **SendEmailAsync()**
   - Generic email sending
   - Parameters: toEmail, subject, body

2. **SendAutoGeneratedPasswordAsync()**
   - Send email with provided password
   - Parameters: toEmail, userName, autoGeneratedPassword

3. **SendAutoGeneratedPasswordWithServiceAsync()**
   - Generate password and send email
   - Parameters: toEmail, userName
