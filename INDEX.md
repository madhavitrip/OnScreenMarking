## 📊 Project Overview

### What is OSM Portal?
A comprehensive on-screen marking system for CBSE Class 12 board exams that allows examiners to digitally evaluate answer sheets with annotation tools, scoring, and comprehensive analytics.

### Key Statistics
- **12 Components** created
- **5 Pages** implemented
- **8 Major Features** developed
- **277.13 KB** bundle size
- **1.64 seconds** build time

### Main Features
1. **Authentication** - Secure login with role selection
2. **Dashboard** - Real-time statistics and progress
3. **Scripts** - Search, view, and mark answer sheets
4. **Reports** - Comprehensive analytics and reporting
5. **Settings** - User preferences and security
